# SeVi Corner Website

## Overview
**SeVi Corner** is a personal website created by Serge Villa. The site showcases interests, skills, and services including English teaching, language learning, coding, and trading. It is designed with a sidebar navigation, animated sections, and responsive layouts for desktop and mobile devices.

---

## Features

- **Sidebar Navigation**: Fixed sidebar with animated slide-in effect.  
- **Animated Content**: Sections and headings use smooth entrance animations.  
- **Responsive Design**: Flexbox-based layout adjusts for smaller screens.  
- **About Me Section**: Highlights teaching experience, strengths, hobbies, and interests.  
- **Message Submission Form**: Text area for user messages with a character limit.  
- **Consistent Styling**: Color-coded banners, highlighted keywords, and visually appealing typography.

---

## Pages

1. **Home** – Introduction with animated welcome lines.  
2. **About Me** – Personal introduction, teaching experience, strengths, hobbies.  
3. **English (ESL)** – Materials and content for English learning.  
4. **Other Languages** – Information about learning other languages.  
5. **Coding Projects** – Portfolio of programming projects.  
6. **Trading Ideas** – Insights and experiences in trading.  
7. **Projects** – Showcases miscellaneous projects.  
8. **Contact** – Form for visitors to get in touch.
